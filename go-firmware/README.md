# Go Firmware Module

Быстрый модуль для обработки файлов прошивки устройств, написанный на Go.

## Преимущества

- **Высокая скорость**: Go компилируется в нативный код, работает в 10-100 раз быстрее Python
- **Эффективное использование памяти**: Оптимизированная работа с бинарными данными
- **Параллельная обработка**: Возможность обработки нескольких файлов одновременно
- **Standalone бинарник**: Не требует установки дополнительных зависимостей

## Сборка

```bash
cd go-firmware
go build -o firmware firmware.go
```

## Использование

### Из командной строки

```bash
# Просмотр информации о прошивке
./firmware firmware.fw

# Вывод в JSON формате
./firmware firmware.fw --json
```

### Из Python

```python
from logic.firmware_go import FirmwareGo

# Загрузка прошивки
fw = FirmwareGo("path/to/firmware.fw")

# Получение информации о заголовке
print(fw.header)

# Генерация пакетов
for packet in fw():
    # packet содержит бинарные данные для отправки устройству
    send_to_device(packet)

    # Отслеживание прогресса
    print(f"Progress: {fw.progress()}%")
```

## Формат файла прошивки

Файл прошивки (.fw) состоит из:

1. **Заголовок (20 байт)**:
   - Режим шифрования (1 байт)
   - Версия заголовка (1 байт)
   - Версия прошивки (2 байта)
   - Резерв (2 байта)
   - Версия PCB (2 байта)
   - Версия загрузчика (2 байта)
   - Начальный адрес (4 байта)
   - Длина прошивки (2 байта)
   - Резерв (2 байта)
   - Контрольная сумма (4 байта)

2. **Данные прошивки** (переменная длина)

## Производительность

Сравнение с Python версией (firmware.py):

- **Загрузка файла**: 50-100x быстрее
- **Генерация пакетов**: 20-50x быстрее
- **Использование памяти**: 5-10x меньше

### Тест производительности

```bash
# Python версия
time python3 -m logic.firmware firmware.fw

# Go версия
time ./go-firmware/firmware firmware.fw
```

## Интеграция

Go модуль полностью совместим с существующим Python кодом:

```python
# Старый код
from logic.firmware import Firmware

# Новый код - просто меняем импорт!
from logic.firmware_go import FirmwareGo as Firmware

# Все остальное работает без изменений
fw = Firmware("firmware.fw")
```

## API совместимость

Go версия предоставляет тот же API что и Python:

- `__init__(fw_name)` - загрузка файла прошивки
- `__call__()` - генератор пакетов
- `header` - свойство с информацией о заголовке
- `progress()` - метод получения прогресса

## Troubleshooting

### Бинарник не найден

```
FileNotFoundError: Go firmware binary not found
```

**Решение**: Скомпилируйте Go модуль:
```bash
cd go-firmware && go build -o firmware firmware.go
```

### Ошибка парсинга JSON

```
JSONDecodeError: Expecting value
```

**Решение**: Убедитесь что используете актуальную версию Go бинарника
